Replaces default token bars with the custom Lancer stat hub. Toggling this requires a reload (you will be prompted on save). Disabled when Bar Brawl is active.

Per-Token Defaults

Defaults applied when a token doesn't have its own override. Per-token settings live in the Resources tab of the token configuration sheet.

Newly placed tokens (and any token without an explicit override) will not show the stat bar hub.

Only display the stat bar hub while the token is part of an active combat encounter.

Override the height of each bar row. Leave blank for the automatic value (scales with grid size).

Visibility Modes

Who can see the stat bar hub. All uses the normal hover/control/target rules. Owners only hides the bars from non-owning players. None hides them entirely (controlled tokens still show for the owner).

Bulk Apply

Writes the per-token default values above onto every Lancer token on the active scene, replacing any existing per-token overrides.
