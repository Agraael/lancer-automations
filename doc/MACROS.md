# Macros

[← Back to the README](../README.md)

Lancer Automations ships a compendium of ready-made macros, the **`L.A -`** set. Drag one to the hotbar and run it on the selected token. Each runs the matching feature documented in the other guides.

## Actions

| Macro | What it does |
|-------|--------------|
| Skirmish | Attack with a single weapon. |
| Barrage | Attack with up to two weapons. |
| Ram | Melee ram; knocks the target Prone. |
| Grapple | Grapple, Break Free, or End Grapple. |
| Brace | Brace, as a reaction. |
| Overwatch | Trigger an Overwatch skirmish. |
| Lock On | Lock On a target. |
| Disengage | Disengage for the turn. |
| Bolster | Grant Bolster to an ally. |
| Aid | Aid an adjacent ally. |
| Fragment Signal | Apply Impaired (or Slow + Impaired) on a tech hit. |
| Invade | Run a tech Invade. |
| Search | Search for hidden characters. |
| Scan | Scan a target. |
| Hide | Become Hidden. |
| Shut Down | Shut down. |
| Boot Up | Boot up from Shut Down. |
| Eject | Eject the pilot from the mech. |
| Dismount | Dismount the pilot from the mech. |
| Handle | Drag or lift an adjacent object or character. |
| Interact | Manipulate a nearby object. |
| Squeeze | Count as one Size smaller, and Prone. |

## Tools

| Macro | What it does |
|-------|--------------|
| Knockback | Push or pull tokens a set distance. |
| Throw Weapon | Throw a weapon onto the field as a token. |
| Pickup Weapon | Pick a thrown weapon back up. |
| Deploy Item | Place one of the actor's deployables. |
| Reinforcement | Set up units to arrive later in combat. |
| Reactor Meltdown | Start a reactor-meltdown countdown. |
| Reactor Explosion | Detonate the meltdown now. |
| Suicide | Drop the token to 0 structure, stress, and HP. |
| Rest | Open the rest / repair dialog. |
| Downtime | Run a downtime activity. |
| Add Reserve | Add a pilot reserve or bonus. |
| Reload One Weapon | Reload a single weapon. |
| End Active Item | End an active (toggled) item. |
| Choice Menu | Build and send a choice or vote card. |
